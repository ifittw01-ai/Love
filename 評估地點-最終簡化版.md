## ✅ 只需要 4 個欄位！

---

## 📊 你的 Google Sheet 應該長這樣

### **頁簽名稱：評估地點**

```
┌─────┬──────┬────────────────────────────────────────┬──────────┐
│  A  │   B  │                   C                    │    D     │
├─────┼──────┼────────────────────────────────────────┼──────────┤
│ ID  │ 國家 │            地點描述                     │ 是否啟用 │
├─────┼──────┼────────────────────────────────────────┼──────────┤
│ tw1 │ TW   │ 11/25 星期一 晚上7:00 台北市信義區...  │   是     │
│ tw2 │ TW   │ 11/27 星期三 下午2:00 台中市西屯區...  │   是     │
│ tw3 │ TW   │ 11/29 星期五 晚上7:00 高雄市前鎮區...  │   是     │
│ my1 │ MY   │ 12/1 Monday 7pm Kuala Lumpur...        │   是     │
│ my2 │ MY   │ 12/3 Wednesday 2pm KLCC...             │   是     │
│ my3 │ MY   │ 12/5 Friday 7pm Petaling Jaya...       │   是     │
└─────┴──────┴────────────────────────────────────────┴──────────┘
```

---

## 📋 4 個欄位

### **第1列 (A)：ID**
- 唯一識別碼
- 範例：`tw1`, `tw2`, `my1`, `my2`

### **第2列 (B)：國家**
- 只能填 `TW` 或 `MY`（必須大寫）
- 台灣 = `TW`
- 馬來西亞 = `MY`

### **第3列 (C)：地點描述**
- 完整的活動信息（日期、時間、地址）
- 這是用戶看到的內容

### **第4列 (D)：是否啟用**
- `是` = 顯示
- `否` = 隱藏
- **留空 = 顯示**（預設啟用）

## ⭐ 如何調整顯示順序？

### **超級簡單！直接拖動行！**

在 Google Sheet 中：
1. 選中整行（點擊行號）
2. 拖動到你想要的位置
3. 完成！

**範例**：
```
如果想讓 tw3 顯示在第一位
只需要把 tw3 那一行拖到 tw1 前面
```

### **操作示意：**
```
原本：
tw1 → 第一個顯示
tw2 → 第二個顯示
tw3 → 第三個顯示

拖動後：
tw3 → 第一個顯示  ⬅️ 拖到這裡
tw1 → 第二個顯示
tw2 → 第三個顯示
```
