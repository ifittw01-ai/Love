## ⚙️ 欄位詳細說明

### **第1列：ID**
- **格式**：文字
- **建議**：使用國家前綴 + 數字（例如：tw1, tw2, my1, my2）
- **作用**：唯一識別碼，用於表單提交
- **必填**：是

### **第2列：國家**
- **格式**：文字
- **值**：只能是 `TW` 或 `MY`（必須大寫）
- **作用**：根據用戶選擇的國家顯示對應地點
- **必填**：是

### **第3列：順序**
- **格式**：數字
- **值**：1, 2, 3, 4...
- **作用**：控制下拉選單中的顯示順序（數字越小越前面）
- **必填**：否（如果不填，會排在最後）

### **第4列：地點描述**
- **格式**：文字
- **內容**：完整的活動信息（日期、時間、地點）
- **作用**：顯示給用戶看的完整信息
- **必填**：是

### **第5列：是否啟用**
- **格式**：文字
- **值**：
  - ✅ 啟用：`是`, `yes`, `YES`, `true`, `TRUE`, `1`
  - ❌ 禁用：`否`, `no`, `NO`, `false`, `FALSE`, `0`, 或留空
- **作用**：控制此地點是否顯示在網站上
- **必填**：是

## 💡 使用技巧

### **1. 調整顯示順序**
只需要修改「順序」欄位的數字，網站會自動按數字從小到大排序

**範例**：
```
如果想把 tw3 移到第一位，把順序改成 0 或 0.5
```

### **2. 臨時隱藏某個地點**
將「是否啟用」改成「否」，該地點就不會顯示在網站上

**範例**：
```
tw4 的活動還沒確定 → 是否啟用 = 否
```

### **3. 添加新地點**
在最後一行添加新的數據即可

**範例**：
```
tw5    TW    5    12/10 星期日 下午2:00~4:00 台南市...    是
```

### **4. 測試地點**
可以先設為「否」，測試完再改成「是」

## ✅ 檢查清單

設置完成後，請確認：

- [ ] 第一行是標題行（ID | 國家 | 順序 | 地點描述 | 是否啟用）
- [ ] 每個 ID 都是唯一的
- [ ] 國家欄位只有 TW 或 MY（大寫）
- [ ] 順序欄位是數字
- [ ] 地點描述包含完整信息（日期、時間、地址）
- [ ] 是否啟用欄位填了「是」或「否」
- [ ] 至少有一個地點是啟用狀態（是）

## 🧪 測試方法

### **測試台灣地點：**
1. 打開網站
2. 選擇國家：台灣
3. 查看「評估地點」下拉選單
4. 應該只顯示「國家=TW」且「是否啟用=是」的地點
5. 順序應該按照「順序」欄位排列

1. 選擇國家：馬來西亞
2. 查看「評估地點」下拉選單
3. 應該只顯示「國家=MY」且「是否啟用=是」的地點

## ⚠️ 常見錯誤

### ❌ 錯誤 1：國家代碼錯誤
```
錯誤：tw, Tw, taiwan
正確：TW
```

### ❌ 錯誤 2：順序不是數字
```
錯誤：一, 第一, first
正確：1
```

### ❌ 錯誤 3：是否啟用值錯誤
```
錯誤：開啟, enable, 顯示
正確：是 或 yes 或 1
```

### ❌ 錯誤 4：沒有標題行
```
錯誤：第一行直接是數據
正確：第一行必須是：ID | 國家 | 順序 | 地點描述 | 是否啟用
```

## 🎨 Google Sheet 美化建議（可選）

1. **第一行（標題行）**：
   - 背景色：淺灰色
   - 字體：粗體
   - 凍結窗格（讓標題行始終可見）

2. **國家欄位**：
   - TW 的行：淺藍色背景
   - MY 的行：淺綠色背景

3. **是否啟用欄位**：
   - 「是」：綠色文字
   - 「否」：紅色文字

4. **自動對齊**：
   - ID、國家、順序、是否啟用：置中對齊
   - 地點描述：靠左對齊

7. 修改成你實際的活動信息
