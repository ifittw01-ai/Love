   # 🚀 AI大腦回春系統 - 全球版

一個支援多語言、多國家的客戶報名系統，具備自動化郵件通知功能。

## ✨ 功能特色

- 🌍 **多語言支援**：繁體中文、簡體中文、英文
- 🌏 **多國家支援**：台灣、馬來西亞（可擴展）
- 🔄 **自動語言切換**：根據用戶瀏覽器語言自動切換
- 📊 **Google Sheet 後台**：所有數據自動儲存
- 📧 **自動郵件通知**：
  - 推廣人員收到新客戶通知
  - 客戶收到確認郵件
- 🎯 **推廣代碼系統**：追蹤每位推廣人員的客戶來源
- 📍 **動態地點選項**：根據選擇的國家自動載入對應的活動場次

## 📋 系統架構

### 前端（GitHub Pages）
- `index.html` - 主頁面
- `script.js` - 前端邏輯
- `translations.js` - 多語言翻譯
- `styles.css` - 樣式表

### 後端（Google Apps Script）
- `google-apps-script-updated.js` - 後端邏輯
- 處理表單提交
- 發送郵件通知
- 提供動態數據 API

### 資料庫（Google Sheet）
**3 個頁簽：**
1. **推廣人員** - 推廣代碼與郵箱對照表
2. **客戶報名記錄** - 客戶報名資料
3. **評估地點** - 活動場次列表

## 🚀 快速開始

### 1. 設置 Google Sheet
1. 創建新的 Google Sheet
2. 創建 3 個頁簽：推廣人員、客戶報名記錄、評估地點
3. 複製 Sheet ID

詳細說明：參考 `Google-Sheet-最終結構說明.md`

### 2. 部署 Google Apps Script
1. 在 Google Sheet 中打開 Apps Script
2. 複製 `google-apps-script-updated.js` 的內容
3. 修改 `SPREADSHEET_ID` 和 `DEFAULT_EMAIL`
4. 部署為網頁應用程式
5. 複製部署 URL

詳細說明：參考 `完整部署指南.md`

### 3. 更新網站代碼
1. 修改 `script.js` 中的 `GOOGLE_SCRIPT_URL`
2. 提交到 GitHub
3. 等待 GitHub Pages 更新

### 4. 測試
訪問網站並提交測試表單，檢查：
- 「客戶報名記錄」頁簽是否收到資料
- 推廣人員是否收到郵件
- 客戶是否收到確認郵件

## 🎯 使用場景

### 推廣人員
1. 分享推廣連結：`網站URL/?ref=你的代碼`
2. 當有客戶報名時，自動收到郵件通知
3. 在 Google Sheet 中查看客戶資料

### 管理員
1. 在 Google Sheet「推廣人員」頁簽中管理推廣代碼
2. 在「評估地點」頁簽中管理活動場次
3. 在「客戶報名記錄」頁簽中查看所有客戶資料
4. 調整地點順序：直接拖動行即可

### 客戶
1. 訪問網站（可能是推廣人員分享的連結）
2. 選擇語言（或自動切換）
3. 選擇國家
4. 選擇活動場次
5. 填寫表單並提交
6. 收到確認郵件（包含 LINE 社群連結）

## 📧 郵件功能

### 推廣人員通知郵件
- 主旨：`🎯 新客戶報名通知 - [客戶姓名]`
- 內容：客戶完整資訊、建議聯繫方式

### 客戶確認郵件
- 主旨：`感謝您報名「AI大腦回春系統」`
- 內容：歡迎訊息、活動資訊、LINE 社群連結

## 🔄 更新系統

### 更新活動場次
1. 打開 Google Sheet
2. 切換到「評估地點」頁簽
3. 新增、修改或刪除資料
4. 拖動行來調整順序
5. 完成！（無需部署）

### 更新推廣人員
1. 打開 Google Sheet
2. 切換到「推廣人員」頁簽
3. 新增、修改或刪除資料
4. 完成！（無需部署）

### 更新網站內容
1. 修改相應的檔案
2. 提交到 GitHub
3. GitHub Pages 會自動更新（1-2 分鐘）
