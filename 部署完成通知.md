## 📝 接下来您需要做什么

### 方案 A：如果是第一次部署（推荐）

**步骤 1：创建 Google Sheet**
1. 前往 https://sheets.google.com
2. 创建新的 Sheet
3. 创建 3 个页签：
   - 推广人员
   - 客户报名记录
   - 评估地点
4. 复制 Sheet ID

**步骤 2：使用配置向导**
1. 在浏览器中打开：`https://ifittw01-ai.github.io/Love/config-wizard.html`
2. 按照向导步骤填写信息
3. 复制生成的配置代码

**步骤 3：部署 Google Apps Script**
1. 在 Google Sheet 中：扩展功能 → Apps Script
2. 粘贴配置向导生成的代码
3. 部署为网页应用程序
4. 复制部署 URL

**步骤 4：更新网站配置**
1. 在本地打开 `script.js`
2. 在第 326 行更新 `GOOGLE_SCRIPT_URL`
3. 保存文件

**步骤 5：推送更新**
运行 `快速部署脚本.bat` 或手动执行：
```bash
git add script.js
git commit -m "Update Google Apps Script URL"
git push origin master
```

## 🆘 如果遇到问题

### 1. 网站无法访问
- 等待 3-5 分钟（GitHub Pages 需要时间更新）
- 清除浏览器缓存（Ctrl+F5）
- 检查 GitHub Pages 设置

### 2. 表单提交失败
- 使用 `test-connection.html` 测试连接
- 检查浏览器控制台（F12）
- 查看 Apps Script 执行记录

### 3. 没有收到邮件
- 检查垃圾邮件文件夹
- 确认推广代码和邮箱对应关系
- 查看 Google Sheet「推广人员」页签

### 4. 评估地区没有选项
- 使用 `test-connection.html` 测试地点加载
- 检查「评估地点」页签的数据格式
- 确认国家代码为大写（TW/MY）
