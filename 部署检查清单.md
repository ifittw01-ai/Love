## 第三步：部署 Google Apps Script

### 3.1 打开 Apps Script 编辑器
1. 在 Google Sheet 中，点击：**扩展功能** → **Apps Script**
2. 会打开一个新的标签页

### 3.2 复制代码
1. 在项目中找到 `google-apps-script-updated.js` 文件
2. 复制**全部内容**
3. 回到 Apps Script 编辑器
4. **删除**默认的代码
5. **粘贴**复制的代码

### 3.3 修改配置

**修改第 22 行：**
```javascript
const SPREADSHEET_ID = '你的Sheet_ID';  // ⚠️ 改成你在第一步复制的ID
```

**修改第 24 行：**
```javascript
const DEFAULT_EMAIL = 'your-email@gmail.com';  // ⚠️ 改成你的默认邮箱
```

### 3.4 保存
点击 **💾 保存专案** 按钮（或按 `Ctrl+S`）

### 3.5 部署为网页应用程序

#### 步骤 A：新增部署作业
1. 点击右上角 **部署** → **新增部署作业**
2. 点击「选择类型」旁边的 **齿轮图标 ⚙️**
3. 选择 **网页应用程序**

#### 步骤 B：设置部署
- **说明**：`客户表单提交系统 v1`
- **执行身份**：选择 **我**
- **存取权限**：选择 **所有人**

#### 步骤 C：授权
1. 点击 **部署** 按钮
2. 第一次会要求授权：
   - 点击 **审查权限**
   - 选择你的 Google 账号
   - 会出现「此应用程序未经验证」警告
   - 点击 **进阶**
   - 点击 **前往（不安全）**
   - 点击 **允许**

#### 步骤 D：复制 URL
部署成功后，会显示「网页应用程序」URL，类似：
```
https://script.google.com/macros/s/AKfycbXXXXXXXXXXXX.../exec
```

**⚠️ 非常重要：复制这个 URL！**
